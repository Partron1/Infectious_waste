# Infectious_waste Overview
This project focus of analyzing a hospital's infectious waste generation using five-year historical data

## Dashboard Summary
The dashboard provides a comprehensive overview of infectious waste generation and collection activities over the year. A total of **4.8 metric tons (4,800 kg)** of infectious waste was generated, managed through **296** collection events, resulting in **751** waste bags collected. The data reveals clear seasonal patterns, quarterly variations, and operational insights that can inform optimization of collection schedules and resource allocation.
